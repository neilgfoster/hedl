# Adopters

Projects that have adopted and are actively using Hedl. Using Hedl and not listed
here? Open a PR adding a row — adopters maintain this file by hand.

Hedl is built using its own workflow: the completion gate, phase discipline, and
adversarial review govern every change to Hedl itself. The dogfood is the proof,
and the reason this list starts with a single row.

## Adding yourself

Add one row with: project name (link optional), the month you adopted, the tier you
run (Gate-only / Lightweight / Team), and a short note on how you use it. Keep it to
one line per project.
